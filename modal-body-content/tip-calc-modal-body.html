Summary

Material Tip Calculator is an Android application that calculates tip and new total based on the subtotal, tip, and number of people that you input for use at restaurants and other places.

What I did

  • Wrote application in Java and styled with XML using Android Studio
  • Created Google Play Developer account and published to the Google Play Store

Features

  • Slider to select tip percentage
  • Split bill amongst multiple people if desired
  • Material Design everywhere from the app icon to the colored status bar
  • Extremely light-weight so the interface is snappy and the app doesn't use any resources in the background
