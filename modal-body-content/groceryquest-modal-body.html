Summary

GroceryQuest is a web application enabling people to create and organize grocery lists by store then sort items by their location within that store to optimize their shopping experience.

What I did

Worked with 1 other student to build the front-end of the application:

  • Designed and implemented an Angular 2 front-end
  • Utilized Bootstrap 3 to make application responsive across all screen sizes from phone to desktop
  • Completely integrated with a back-end database over http using a RESTful API

Features

  • Create an infinite number of grocery lists
  • Autocomplete when typing in new items from an expansive list of common items
  • Sort items in lists manually, alphabetically, or by their location in the store
  • View heatmap of how frequently you've shopped over the past year
  • Spent 4 weeks on human-centered design to develop features and a user-interface that revolve around the user's needs
