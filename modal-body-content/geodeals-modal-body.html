Summary

GeoDeals is a web application enabling people to share the great deals they find in their daily lives with their local community and share their opinion on deals in their area for everyone's mutual benefit.

What I did

Worked with 1 other student to build the back-end of the application:

  • Designed and implemented a MySQL database consisting of 13 tables with referential integrity
  • Launched and configured an AWS EC2 instance with a LAMP stack to host the application on the internet with a custom domain
  • Used the Slim Framework to expose a RESTful API to enable the Angular 2 front-end to interact with the back-end database

Features

  • Identifies users location using the browser if they enable location services or their IP address if they don't
  • Uses the Yelp Fusion API to retrieve information about nearby business according to the user's current location and an optional search query
  • Event driven MySQL master-slave database replication
  • Email verification: Email sent to user when they register and they must click embedded link before they can log in
  • User data tracking and logging of operating system, browser, and IP address for all http requests
  • Data cleansing with Cron and PHP scripts
